# george's goodreads

좋은 콘텐츠를 한곳에 모아 읽기 쉽게 정리하는 큐레이션 웹앱입니다.  
해외 Article 번역, 유튜브 영상 분석 리포트, 국내 Article 요약을 카테고리별로 탐색할 수 있습니다.

## 방문자 안내

- 서비스 주소: **[https://george-goodread.vercel.app](https://george-goodread.vercel.app)**
- 상단에서 카테고리/타입을 골라 볼 수 있습니다.
- 카드 클릭 시 콘텐츠 상세 페이지로 이동해 전체 본문과 댓글을 볼 수 있습니다.
- 댓글은 이름, 메시지, 비밀번호를 입력해 작성합니다.

## 콘텐츠 타입

- `해외 article`
  - 링크 기반 번역/정리본입니다.
- `유튜브`
  - 요약 markdown이 표시되고, 상세 페이지 상단에서 영상이 임베드됩니다.
- `국내 article`
  - 요약과 리다이렉트 링크 중심으로 정리됩니다.

## 카테고리

- `ai, 바이브코딩`
- `크립토, 트레이딩, 투자`
- `경제, 사회`
- `삶, 건강`

## 관리자 페이지(운영자용)

- 관리자 페이지: `/admin`
- 새 글 등록/수정/삭제 지원
- 입력한 콘텐츠는 GitHub 기반 저장소로 영구 저장되어 재배포 시에도 유지됩니다.
- 운영 비밀키는 환경변수로 관리됩니다.

## 핵심 정책

- Markdown 형식으로 콘텐츠를 제공하며, 주요 링크는 본문에서 바로 열릴 수 있게 처리합니다.
- 제목 클릭, 경로 이동은 공개 URL(`https://george-goodread.vercel.app/item/:id`)를 기반으로 동작합니다.
